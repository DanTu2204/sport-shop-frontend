# 🏅 Sport Shop - Frontend Storefront

Chào mừng bạn đến với repository Frontend của dự án **Sport Shop**! Đây là giao diện người dùng trực quan, hiện đại và phản hồi nhanh (responsive), được thiết kế để mang lại trải nghiệm mua sắm đồ thể thao tốt nhất.

## 🚀 Tổng quan dự án

Dự án này là một phần của hệ thống thương mại điện tử chuyên biệt về đồ thể thao, bao gồm:
- **Frontend**: Ứng dụng Single Page Application (SPA) mạnh mẽ dành cho khách hàng mua sắm.
- **Backend & Admin Panel**: Hệ thống API phục vụ dữ liệu và giao diện quản trị (SSR) dành cho người quản lý.

## 🛠 Công nghệ sử dụng

Ứng dụng được xây dựng trên bộ công nghệ hiện đại:
- **ReactJS**: Thư viện JavaScript hàng đầu để xây dựng giao diện người dùng.
- **Vite**: Công cụ build siêu nhanh, tối ưu hóa quá trình phát triển.
- **Axios**: Xử lý các yêu cầu HTTP đến Backend API.
- **React Router DOM**: Quản lý điều hướng và định tuyến trong ứng dụng.
- **Lucide React & FontAwesome**: Bộ icon chuyên nghiệp.
- **Vanilla CSS & Bootstrap**: Tối ưu hóa giao diện và trải nghiệm người dùng.

## ✨ Tính năng chính

- **Trang chủ hiện đại**: Carousel Banner động, hiển thị sản phẩm mới nhất và sản phẩm nổi bật.
- **Danh mục sản phẩm**: Lọc và xem sản phẩm theo từng danh mục thể thao chuyên biệt.
- **Authentication**: Hệ thống Đăng ký/Đăng nhập bảo mật.
- **Quản lý Tài khoản**: Xem thông tin cá nhân và lịch sử hoạt động.
- **Giỏ hàng & Thanh toán**: Luồng mua hàng mượt mà, tích hợp nhiều phương thức thanh toán.
- **Tích hợp Admin**: Tự động chuyển hướng người quản trị sang trang quản lý chuyên sâu ở Backend.

## 📦 Hướng dẫn cài đặt và Chạy cục bộ

Để chạy dự án này trên máy tính của bạn, hãy làm theo các bước sau:

1. **Clone repository**:
   ```bash
   git clone <link-repo-cua-ban>
   cd <thu-muc-frontend>
   ```

2. **Cài đặt các thư viện (dependencies)**:
   ```bash
   npm install
   ```

3. **Cấu hình môi trường**:
   Mở file `src/api/axios.js` và cập nhật `baseURL` trỏ về địa chỉ Backend của bạn (localhost hoặc production).

4. **Chạy ứng dụng trong môi trường development**:
   ```bash
   npm run dev
   ```
   Ứng dụng sẽ chạy tại: `http://localhost:5173`
